INSTALACIÓN DE GIT
1.instalar Git (control de versiones). https://git-scm.com/ ----->https://git-scm.com/download/win (64-bit Git for Windows Setup.)(Marcar todo los campos )
2.Comprobamos en terminal si tenemos instalado Git correctamente: git --version
Si está bien instalado nos debe aprecer : git version 2.43.0.windows.1, al pulsar enter.
3. Abrir una ventana (sin proyectos). Luego ir a las ... de la barra de herramientas y abrir terminal-> new terminal. Comprobamos que esta insatalado git.(usamos el siguiente tutorial).
CREACIÓN DE REPOSITORIO REMOTO

2.Le damos al botón "New" de color verde de la izquierda y comenzamos a crear un repositorio remoto.
